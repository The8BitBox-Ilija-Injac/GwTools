# GwTools
A singe Setup file for windows, that installs the latest Grease Weazle and Fluxengine commandline tools on Windows and sets the evnironment variables of the toolset for easy access via the commandline on Windows!
# Important
There is no source code available here, just the Installer itself. As the included tools change in the future, the installer will be updated.
